To deploy on GitHub Pages:
  1. Create a GitHub account at https://github.com
  2. Create a new repository (e.g., "portfolio")
  3. Upload all files keeping the folder structure
  4. Go to Settings → Pages → Branch: main → Save
  5. Your site will be live at: https://yourusername.github.io/portfolio

To deploy on Netlify (easiest):
  1. Go to https://netlify.com and sign up free
  2. Drag and drop the entire portfolio folder onto the Netlify dashboard
  3. Get an instant live URL!
